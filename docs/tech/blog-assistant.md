# 终端博客助手

```powershell
cd D:\my-blog
npm run blog
```

飞书优先。完整说明见 [飞书写作 → 一键发布](feishu-publish.md)。

## 菜单

| 键 | 功能 |
|----|------|
| S | 飞书配置 / 登录（首次必做） |
| 1 | 飞书文档 → 博客（可推送） |
| 2 | 创建飞书博客台账 |
| 3 | 从台账一键发布 |
| 4 | 重同步已绑定文档 |
| 5 | 本地预览 |
| 6 | 外站导出 |
| 7 | 构建 / Git 推送 |
| 8 | 列表 |
| 9 | 默认参数 |

直接回车使用 `[默认值]`。默认保存在本机 `sync/assistant-defaults.json`。
