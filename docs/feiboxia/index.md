## 怎么用

1. **写作台**：在飞书云文档写随笔（编辑即预览）  
2. **稿件箱**：台账登记文档链接、导航栏目、状态  
3. **发布台**：本机执行一键发布，或打包队列交给 GitHub Actions  

```powershell
cd D:\my-blog
npm run feiboxia:ship
```
