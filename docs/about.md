## 权限说明

| 级别 | 能力 |
|------|------|
| 作者（本机） | 编辑 Markdown、私密稿、`npm run publish:*` 一键/批量发布 |
| 访客（公开站） | 浏览公开文章、Giscus 评论 |

私密内容仅保存在本地 `docs/private/`，不会进入公开构建。

## 评论

文章页底部为 **Giscus**（GitHub Discussions）。首次使用请：

1. 仓库 Settings → Features → 勾选 **Discussions**
2. 打开 [giscus.app](https://giscus.app/zh-CN)，选仓库 `duniang818/wahaha`，生成配置
3. 把 `data-category-id` 填进 `overrides/partials/comments.html` 后推送
