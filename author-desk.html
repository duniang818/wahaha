作者发布台（L1 权限 · 仅本机）

此页不部署到公网。访客无法使用发布能力。密钥只放在本机 .env。

单篇发布

npm run publish:one -- welcome --to wechat,xhs,csdn,zhihu
npm run publish:one -- welcome --dry-run

批量发布

npm run publish:batch -- --tag 旅行 --to xhs,wechat --dry-run
npm run publish:batch -- --tag 旅行 --to xhs

权限规则

  • docs/private/** 或 visibility: private → 禁止外发
  • draft: true → 禁止外发
  • platforms: 白名单限制可发平台

本地预览博客

npm run dev
# http://127.0.0.1:8000/
