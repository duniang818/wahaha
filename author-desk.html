作者发布台（L1 · 仅本机）

访客看不到本页。密钥只放本机 .env。

推荐工作流：飞书写作 → 一键上博客 
lark-cli auth login
npm run from-feishu -- "https://xxx.feishu.cn/docx/TOKEN" --push
# 同时发外站：
npm run from-feishu -- "URL" --push --also wechat,xhs,csdn,zhihu

飞书 → 博客

npm run from-feishu -- "<飞书文档URL>"
npm run from-feishu -- "<URL>" --slug my-post --dir blog/posts --push
npm run from-feishu -- --list

博客 Markdown → 外站

npm run publish:one -- welcome --to wechat,xhs,csdn,zhihu
npm run publish:batch -- --tag 旅行 --to xhs --dry-run

权限

  • docs/private/** / visibility: private → 禁止外发
  • draft: true → 禁止外发
  • 加 --private 可把飞书文写入私密区（仅本机）

npm run dev
