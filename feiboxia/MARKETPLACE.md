# 飞博虾 · 飞书应用市场上架材料

## 应用信息（建议填报）

| 项 | 内容 |
|----|------|
| 应用名称 | 飞博虾 |
| 英文名 | FeiBoXia |
| 一句话介绍 | 在飞书写文章，台账点一下就发到个人博客 |
| 详细介绍 | 飞博虾帮助个人作者用飞书云文档写作，用多维表格管理栏目、定时与发布状态；一键或定时同步到 GitHub Pages / MkDocs 博客对应导航目录。免费使用，面向个人知识博主。 |
| 分类 | 效率工具 / 内容创作 |
| 价格 | 免费 |
| 适用 | 个人用户、小团队 |

## 需要开通的权限（scopes）

- 云文档：读/写（拉取正文）
- 多维表格：读/写、工作流（台账与按钮/自动化）
- 可选：消息（安装欢迎卡、发布结果通知）

## 审核说明要点

1. **数据流**：用户文档仅在用户授权下由服务读取，写入用户自己绑定的博客仓库。  
2. **密钥**：GitHub Token / webhook secret 仅存租户配置，不上报第三方分析。  
3. **免费**：安装即用；可后续增值但不强制。  
4. **公网**：提供稳定 HTTPS 回调（推荐云主机或 Serverless）。
